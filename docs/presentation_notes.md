# Presentation Notes

## Main Story
Cyclistic annual members and casual riders use the bike-share service differently in frequency, duration, timing, and seasonality.

## Key Talking Points
- Members generate more rides overall
- Casual riders have longer average ride durations
- Members show a strong weekday and commute-like pattern
- Casual riders peak on weekends
- Casual demand rises in warmer months

## Main Recommendation Themes
- Target weekend casual riders with membership conversion campaigns
- Increase campaign intensity during spring and summer
- Position membership as a value product for repeat leisure use, not only commuting
